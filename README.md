
# Portfólio Profissional — Artioli

Este é o repositório do meu **Portfólio Profissional**, desenvolvido como parte do meu plano avançado de estudos em front-end.  
O objetivo deste projeto é apresentar minhas habilidades, experiências e projetos reais de forma moderna, organizada e com design inspirado em interfaces premium.

---

## 🚀 Tecnologias Usadas

- **HTML5**
- **CSS3 (Dark/Light Mode + temas personalizados)**
- **JavaScript Moderno (ES6+)**
- Design responsivo
- Microinterações e animações suaves
- Componentes reutilizáveis

---

## 🎨 Features

### ✔ Interface Premium
Layout baseado em portfolios internacionais, com tipografia forte e estrutura visual elegante.

### ✔ Suporte a múltiplos temas
- Tema **Dark**
- Tema **Light**
- Tema **Creative B** (tema alternativo estilizado)

Gerenciamento de tema salvo em `localStorage`.

### ✔ Seções presentes
- Apresentação
- Experiência profissional
- Formação Acadêmica
- Projetos desenvolvidos
- Contato
- Currículo estilizado (tipo UI/UX resume)

## 🛠 Como rodar localmente

1. Clone o repositório:

```
git clone https://github.com/SEU-USUARIO/portfolio
```

2. Acesse o diretório:
```
cd portfolio
```

3. Abra o arquivo principal:
```
index.html
```

> Não precisa de servidor — é HTML/CSS/JS puro.
